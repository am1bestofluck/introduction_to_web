семинар основы js

код выполняется построчно сверху вниз;  так подгру
женный скрипт <script src ="some.js"> </script> выполнится соответственно свой позиции в документе;
Скрипт можно поставить в <head></head>, но тогда он не сможет работать
с скомпилированным контентом; может взаимодействовать с ним через например теги;  

> Есть решения:

	> старое:

		скрипты ставить перед закрывающим </body>

	> новое:

		 скрипты внутри head но явно указать "грузись позже"( defer) 
		 <head>
		 	<script src = "test.js" defer ></script>
		 </head>

	> редкое:

		 есть ещё аттрибут async; скрипт загружается асинхронно по мере высвобождения ресурсов
